## Features Overview

| Feature | Description |
|---------|-------------|
| **Device Management** | Inventory management with Netbox integration |
| **Template Deployment** | Jinja2-based configuration with variable substitution |
| **Service Stacks** | Group related services with dependency ordering |
| **MOP Engine** | Visual procedure builder with conditional logic |
| **Config Backups** | Automated backups with diff comparison |
| **AI Agents** | LLM-powered automation with tool integration |
| **Multi-Auth** | Local, LDAP, and OIDC authentication |

## Requirements

- Docker and Docker Compose
- 2GB RAM minimum (4GB recommended)
- Network access to managed devices (SSH)
- (Optional) Netbox for device inventory
- (Optional) LLM API keys for AI features
