API Authentication

All API endpoints (except login and webhooks) require JWT authentication.

  1. Obtain a JWT token via POST /api/auth/login
  2. Include the token in the Authorization: Bearer <token> header
  3. Refresh tokens via POST /api/auth/refresh before expiry
