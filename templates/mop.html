YAML MOP Syntax: 
name: "My MOP"
description: "What this mop does"
devices:
  - device1.example.com
  - device2.example.com

steps:
  - name: "Check BGP Neighbors"
    id: check_bgp
    type: check_bgp
    expect_neighbor_count: 4
    compare_to_netbox: true
    on_success: deploy_stack
    on_failure: send_alert

  - name: "Deploy Stack"
    id: deploy_stack
    type: deploy_stack
    stack_id: "my-stack-id"
    on_success: send_success
    on_failure: send_alert

  - name: "Send Success Email"
    id: send_success
    type: email
    to: "ops@company.com"
    subject: "Deployment Complete"
    body: "Deployed at {timestamp}"

  - name: "Send Alert"
    id: send_alert
    type: email
    to: "oncall@company.com"
    subject: "MOP Failed"

Available Step Types:
- check_bgp, check_ping, check_interfaces
- deploy_stack, run_command
- email, webhook
- custom_python, wait

Variables:
Use {timestamp}, {mop_name}, {device_name}, etc.
